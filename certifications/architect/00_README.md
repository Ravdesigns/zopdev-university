# Architect Certification

§ Application-based · Take-home + interview · Named credential

## Outcome

"ZopDev Certified Cloud Cost Architect" — design enterprise governance posture using ZopNight at scale.

## Prerequisites

- Engineer certification (required)
- 6+ months hands-on production experience with ZopNight
- Application + selection (limited cohort)

## Format

```
1. Application form
2. Take-home design exercise (1-week)
3. 45-minute review interview with the editorial board
```

## Application

```
APPLICATION FORM ASKS:
  - Current ZopNight deployment context
  - Cloud estate size + complexity
  - Specific design challenges you've solved
  - Why you want the credential
  - References (1 colleague, 1 customer if applicable)
```

## Take-home design

Pick from 3 design challenges:

```
DESIGN A: Multi-account governance for 200+ accounts across 5 regions
  Cover: RBAC structure, audit log policy, schedule design at scale,
  cost attribution, anomaly response

DESIGN B: Hybrid (cloud + on-prem) FinOps for a regulated industry
  Cover: compliance evidence, audit log policy, integration with
  on-prem systems, attribution

DESIGN C: Cross-organizational ZopNight rollout (MSP-style)
  Cover: multi-tenancy, per-customer isolation, scaling operations
```

Submit a 5-10 page design document with diagrams.

## Interview

45 minutes with the editorial board. Questions:

- Walk through your design
- Defend trade-offs
- Discuss anti-patterns you considered
- Demonstrate understanding of edge cases
- Explain how you'd evolve the design as the org grows

## Pass criteria

- Take-home demonstrates architectural rigor
- Interview shows depth of understanding
- Both required

## Resubmission

If pass criteria not met, candidate can resubmit. The editorial board provides specific feedback on what to address.

## Issuance

- Named credential ("ZopDev Certified Cloud Cost Architect")
- Public registry listing (with the architect's bio)
- Shareable to LinkedIn
- Often appears on consulting / professional services profiles

## Cohort schedule

Quarterly. Approximately 5-10 architects per cohort.

## When to apply

When you've been hands-on with ZopNight at significant scale for 6+ months. The application's quality is part of the evaluation.
