# Certifications

Three tiers. Each validates a specific competency level.

## Tiers

| Tier | Free? | Format | Pass | Issued |
|---|---|---|---|---|
| **Operator** | Yes | 20 MCQs, 30 min, open-book | 80% | Digital badge (Credly) |
| **Engineer** | Free for customers; $99 outside | 40 MCQs + 1 graded lab | 75% on each | Verifiable cert, public registry |
| **Architect** | Application-based | Take-home + interview | Pass/re-submit | Named credential |

## Paths

| Cert | Recommended preparation |
|---|---|
| Operator | Track 0 + Track 1 |
| Engineer | All operator material + Track 2 + Track 4 + Track 5 + Track 6 |
| Architect | All engineer material + Track 3 + extensive production experience |
