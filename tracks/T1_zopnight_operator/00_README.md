# Track 1 — ZopNight Operator

§ T1 · Operator tier · ZopNight blue · No prior ZopNight experience required · ~4.5 hours

---

## What you'll be able to do after this track

Connect AWS, GCP, and Azure accounts to ZopNight. Read the resource list and filter it to anything in the estate. Build a schedule with cron, timezones, presets, and a visual 24-hour grid. Bundle resources into groups with sequenced execution. Set time-bounded overrides with reasons and expiry. Read state-history, configure Slack / Teams / GChat notifications, and query the audit trail.

You will use ZopNight directly throughout. Have a sandbox or trial account ready before starting M1.3.

---

## Who this is for

| Persona | Why this track |
|---|---|
| Platform / DevOps Engineer | The product they will operate daily |
| FinOps Analyst | The surface that supplies cost and savings data |
| Engineering Leader | Enough to ask informed questions of the team using it |

This track assumes you have completed **Track 0 — Cloud Cost Foundations**, or have equivalent domain knowledge. The lessons reference Track 0 concepts (rack rate, CDCR, two-source cost model) without re-teaching them.
