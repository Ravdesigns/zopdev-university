# M1.3 — Build your first schedule

§ T1 · M1.3 · Operator tier · 6 lessons · ~60 min

---

## Module outcome

Create a schedule with name, timezone, start and stop crons. Read and write cron expressions confidently. Verify schedule correctness on the 24-hour weekly grid. Apply the three preset schedules (Business Hours, Peak Hours, Weekend Scale-Down). Attach resources and groups. Read the savings estimator.
