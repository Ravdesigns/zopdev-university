# Track 0 — Cloud Cost Foundations

§ T0 · Operator tier · Free · No product login required · ~5 hours

---

## What you'll be able to do after this track

Read any AWS, GCP, or Azure cloud bill end to end. Explain the FinOps Foundation framework in your own words. Calculate the math of scheduling against commitments. Distinguish rack rate, billing cost, and amortized cost. Define CDCR — Continuous Detect, Continuous Remediation — and explain why "detect and act" beats "detect and ticket."

You will not use ZopNight in this track. The product walkthroughs start in Track 1. This track is the domain knowledge a sharp engineer or analyst needs before any tool helps.

## What this track is NOT

- It is not a sales pitch for ZopNight. Track 1 is. This track works whether you ever buy us or not.
- It is not Terraform / CDK / IaC training. We point to HashiCorp Learn and the AWS CDK docs when those topics come up.
- It is not a deep dive on any single cloud service. The compute / storage / database depths sit in Tracks 2 and 5.

---

## How to read this track

- Each lesson is 5 to 12 minutes. Read straight through or pick what you need.
- Lessons have five parts: **Outcome**, **Concept**, **Demo**, **Hands-on**, **Knowledge check**, and **Apply**. If you skip any part, skip Demo. Don't skip the Knowledge check.
- The Hands-on in this track uses **your own cloud bill**, not the product. Bring last month's invoice.
