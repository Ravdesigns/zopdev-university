# M0.2 — FinOps Foundation principles

## Module outcome

Name the six FinOps principles. Explain the three-phase lifecycle (Inform / Optimize / Operate) and the four levers of cost optimization. Self-assess an organization against Crawl / Walk / Run maturity. Read a FOCUS-formatted billing record without a decoder ring.
