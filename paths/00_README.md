# Role-based Learning Paths

Five curated paths through the curriculum. Each path picks lessons from across tracks for a specific role.

## The five paths

| Path | Audience | Duration | Cert pairing |
|---|---|---|---|
| [Platform / DevOps Engineer](platform_engineer.md) | Hands-on platform engineering | 14 hours | Operator → Engineer |
| [FinOps Analyst](finops_analyst.md) | FinOps practitioners | 12 hours | Operator → Engineer |
| [Engineering Leader](engineering_leader.md) | Eng directors / VPs | 6 hours | (no cert required) |
| [Finance Partner](finance_partner.md) | FP&A / procurement | 5 hours | (no cert required) |
| [Security / Compliance](security_compliance.md) | Security architects / compliance | 10 hours | Architect track |

## How to use a path

1. Open your path's page
2. Follow the lessons in order
3. Skip lessons you've already taken (each is independent)
4. Optionally take the suggested certification
